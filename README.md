# 💸 Widget Hemat Cuan

Widget Hemat Cuan is a simple desktop-based money manager application built using HTML, CSS, JavaScript, and Electron.

This application helps users manage daily finances by tracking income and expenses through a modern, lightweight, and user-friendly interface.

---

## ✨ Features

- 📊 Income & expense management
- 💾 Local data storage
- 🖥️ Native desktop application

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Electron

---

## 📦 Installation

### Download Release

1. Download the latest release
2. Extract the ZIP file if needed
3. Run the `.exe` file

---

## 🚀 Development Setup

Clone the repository:

```bash
https://github.com/KicauOrgspark/Hemat-Cuan-
```

Open the project folder:

```bash
cd widget-keuangan
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm run dev
```

Build the application:

```bash
npm run dist
```
